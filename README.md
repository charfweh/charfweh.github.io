<div align = "center"><p>A single page portfolio made with Bootstrap Studio.</p></div>

## :information_source: Pre-requisites
- Bootstrap Studio License.
- Beginners tutorial heavily recommended

## :inbox_tray: Installation
- Clone the repository.
- Open ``Portfolio.bsdesign`` in Bootstrap studio.
- Export the design to a folder and push to your repository.
